# crypto-page
Project to display cryptocurrency information

## Technologies
- React
- React Query
- Tailwind

## Api
- https://api.coingecko.com/api/v3

## Installation
 
 ```
 git clone https://github.com/kevob1994/crypto-page.git
 
 npm install
 
 npm run dev
 
 ```
 
 ## Consider
 
 - The coingecko api has a request limit because it is free
